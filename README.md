# Food Nutrition Capstone Project
**Team 3 - Capstone 2025**

## Project Description
A comprehensive nutrition and meal planning web application with food database, ingredient tracking, and meal management.

## Features
- Food nutrition database search
- Ingredient library
- Meal planning and recipes
- FDC nutrient data integration
- Responsive web design

## Technology Stack
- PHP 7.4+
- MySQL
- HTML5/CSS3
- JavaScript

## Installation

### Option 1: Web Hosting
Visit: http://foodcapstone.rf.gd

### Option 2: Local Installation
1. Clone this repository
2. Import database schema from `database/schema.sql`
3. Update `config.php` with your database credentials
4. Access via local web server

## Project Structure
- `index.php` - Homepage with search and stats
- `search.php` - Unified search across all data types
- `foods.php` - Food database management
- `ingredients.php` - Ingredient library
- `meals.php` - Meal recipes
- `config.php` - Database configuration
